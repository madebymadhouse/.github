# System Overview

The WokSpec org-profile repository contains:

- public organization profile content
- contributor guidance (CONTRIBUTING, CODE_OF_CONDUCT, SECURITY, DEVELOPMENT)
- reusable workflow templates
- ecosystem documentation

Its job is to explain the organization clearly, not to house product code.
