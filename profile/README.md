# WokSpec

An AI-native platform for people who create things.

Pick a template. Connect what you already use. Ship something real.

| | |
|--|--|
| [**Eral**](https://eral.wokspec.org) | AI that knows your brand. Works in your browser, embeds in any product. |
| [**Dilu**](https://dilu.wokspec.org) | Build and launch products without code. Integrates with Shopify, Stripe, Discord, and more. |
| [**Vecto**](https://vecto.wokspec.org) | AI design studio. Icons, brand kits, and visual assets. |
| [**WokPost**](https://wokpost.wokspec.org) | AI-curated news platform. 20 categories, no noise. |
| [**WokTool**](https://tools.wokspec.org) | 80+ browser tools. Free, no login. |
| [**Chopsticks**](https://chopsticks.wokspec.org) | Open-source Discord bot. Music, moderation, economy, and AI. |
