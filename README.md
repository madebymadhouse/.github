# Mad House Org Defaults

This repository is the organization-level source of truth for Mad House GitHub defaults.

It contains:
- the org profile README shown on the GitHub organization page
- shared community health files like contributing and security policies
- reusable workflow templates for PR quality, secret scanning, commit hygiene, and release flow

If a Mad House repo needs a default PR template, issue template, workflow template, or baseline contributor policy, it should come from here.

Canonical workflow standard:

https://github.com/madebymadhouse/bot-dev-playbook/blob/main/AGENTIC_GIT_WORKFLOW.md
